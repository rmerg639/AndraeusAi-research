# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and inclusive environment for everyone interested in Andraeus AI research.

## Our Standards

### Positive Behavior

- Being respectful and inclusive
- Providing constructive feedback
- Focusing on what's best for the project
- Showing empathy towards others

### Unacceptable Behavior

- Harassment or discriminatory language
- Personal attacks or trolling
- Publishing others' private information
- Spam or off-topic discussions

## Scope

This Code of Conduct applies to:

- GitHub repository interactions
- Issue discussions
- Pull request reviews
- Email communications
- Any project-related spaces

## Enforcement

Violations may result in:

1. Warning
2. Temporary ban
3. Permanent ban

## Reporting

Report violations to: andraeusbeats@gmail.com

All reports will be reviewed and investigated promptly and fairly.
