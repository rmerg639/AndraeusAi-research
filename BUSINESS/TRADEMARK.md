# Trademark Registration Guide

## Registering "Andraeus AI" Trademark

### What to Trademark

| Mark | Type | Priority |
|------|------|----------|
| ANDRAEUS AI | Word mark | HIGH |
| ANDRAEUS | Word mark | HIGH |
| Logo (if created) | Device mark | MEDIUM |

---

## Step 1: Search Existing Trademarks

**Before applying, search for conflicts:**

1. Go to: https://search.ipaustralia.gov.au/trademarks
2. Search for: "ANDRAEUS"
3. Search for: "ANDRAEUS AI"
4. Check no similar marks exist in Class 42

---

## Step 2: Choose Classes

### Recommended Classes

**Class 9 - Software**
```
Computer software; downloadable software for artificial intelligence;
machine learning software; software for natural language processing;
computer programs for data processing.
```

**Class 42 - Technology Services**
```
Software as a service (SaaS); artificial intelligence consultancy;
computer programming services; technological research and development;
licensing of computer software; AI research services.
```

**Cost per class:** ~$250 AUD

---

## Step 4: File Application

1. Go to: https://online.ipaustralia.gov.au
2. Create account or login
3. Select "Trade Marks" → "Apply for a trade mark"
4. Enter mark details
5. Select classes
6. Pay fee (~$250 per class)
7. Submit application

---

## Step 5: Timeline

| Stage | Time | Action |
|-------|------|--------|
| Filing | Day 1 | Submit application |
| Examination | 4-6 months | IP Australia reviews |
| Acceptance | 6-8 months | Published for opposition |
| Opposition period | 2 months | Others can oppose |
| Registration | 8-12 months | Trademark registered |
